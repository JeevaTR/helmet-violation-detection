# Helmet Detection using YOLOv8

This project implements a helmet detection system using YOLOv8 and a custom dataset.

## Features
- Helmet detection on images
- YOLOv8 training and inference
- Custom dataset (YOLO format)

## Tech Stack
- Python
- Ultralytics YOLOv8
- OpenCV

## How to Run
```bash
python3 train.py
python3 predict.py
